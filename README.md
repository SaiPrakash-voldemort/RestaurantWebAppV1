# Silver Spoons Restaurant Web App

The **Silver Spoons Restaurant Web App** is a modern, responsive website for a restaurant. It features a dynamic menu, booking system, and user-friendly interface built with React.js and managed with Redux for state management.

## Features
- **Frontend**:
  - Built using React.js with Redux for state management
  - Material UI and Tailwind CSS for a modern design
  - Dynamic menu with categorized items
  - Responsive design (mobile-first approach)

- **Functionality**:
  - Menu items are displayed dynamically from a JSON server
  - Customers can add items to the cart
  - Integrated with Redux for state management, including cart and user data

- **Deployment**:
  - Deployed on GitHub Pages for easy access
